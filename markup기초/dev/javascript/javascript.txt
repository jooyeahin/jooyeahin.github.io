[자바스크립트 개요]
- 정식명칭 ECMAScript(유럽 컴퓨터 제조 협회)
- 브라우저마다 조금씩 다른 스크립트를 사용하나 통합해서 javascript라고 함.
- 동적인 웹페이지 작성을 위해 사용되는 클라이언트 측 프로그램언어
- 모든 웹브라우저에 자바스크립트 해석기가 내장되어 있음.
- 자료형은 가변적이다.

[자바스크립트 활용]
- 이벤트를 사용한 제어
- 요소를 동적으로 추가, 제거, 수정 가능
- 데이터 처리

[코드 오류확인]
크롬브라우저 F12 => console탭에서 오류메세지를 확인함
=>오류가 난 라인을 찾아서 문제점 해결

[기본용어]
- 표현식
  값을 만들어내는 간단한 코드
  ex)10+20
     '안녕하세요'

- 문장
  하나이상의 표현식이 모인 것으로 문장이 모여 프로그램이 구성된다.
  문장의 끝에는 세미콜론(;)을 찍어 문장을 종결시킴
  ex)var sum=10+20;
     alert('합계:'+sum);

- 키워드(예약어)
  자바스크립트에서 사용하고 있는 특별한 단어
  키워드는 이미 사용하고 있는 단어이므로 변수나 함수등의 이름을 작성할 때 주의한다.(사용금지)
  ex) var, alert, false, boolean ...

- 식별자
  자바스크립트에서 변수, 함수등의 이름을 붙일 때 정한 규칙
  *식별자 규칙(반드시 지켜야함)
  1. 키워드를 사용하면 안됨
  2. 숫자로 시작하면 안됨.
  3. 공백안됨
  4. 특수문자안됨(단 _, $는 가능)
  5. 대소문자를 구분함.

  *개발자들이 관례로 지키는 규칙(반드시 지킬 필요는 없으나 지키면 좋음)
  1. 생성자의 이름은 대문자로 시작한다.
  2. 변수, 함수, 메서드의 이름은 소문자로 시작한다.
  3. 여러단어가 이어질 경우 두번째 단어부터 첫글자는 대문자로 처리한다.
  4. 의미있는 단어를 사용한다.
     ex) 숫자 num, 문자 str
         a, b, c등의 의미없는 이름은 사용하지 않는다.
