[선택자 종류]
1. 전체선택자
  모든 요소를 선택함.
  주로 초기화작업을 할 때 사용함.
  브라우저마다 기본으로 설정된 값(특히 여백값)이 다르므로 반드시 필요.

  ex) *{margin:0; padding:0;}
      article안에 있는 모든 요소 선택
      article *{}

2. 요소(태그) 선택자
  요소의 이름으로 선택
  ex) h1{}
      p{}

3. 아이디선택자
  요소에 속성으로 id를 사용할 경우
  id값은 중복되면 안되므로 문서상에서 한번만 써야함
  id는 전체 문서구조상 유일한 구조에 해당되는 것에만 적용하는것이 좋다.
  ex) logo, content, popup, 시멘틱태그를 사용안할 시 큰 그룹의 이름
      스크립트로 특정기능을 부여하기위해 아이디값을 걸어줌.

  #header{}
  #gnb{}

4. 클래스 선택자(가장많이 사용함)
  요소에 속성으로 class를 사용하는 경우
  전체문서의 내용에서 반복적으로 사용가능
  클래스의 값은 여러개를 걸어줄 수 있다.
  주로 공통적으로 처리해야할 영역을 클래스 규칙을 정하여 처리함.

  ex)빨간색 글자 규칙 정하기
  .fontRed{color:red;}
  안쪽여백이 10px인 규칙 정하기
  .p10{padding:10px;}
  .p20{padding:20px;}
  .p30{padding:30px;}
  .p40{padding:40px;}
  .p50{padding:50px;}

5. 자식 선택자
  선택한 부모의 자식을 선택할 때 사용
  주로 동일한 구조가 중첩되어 있을 경우 사용한다.
  부모요소 > 자식요소
  ex) div > p
  div
    p
    section
      p
6. 후손선택자
  선택한 부모의 후손(자식포함)을 선택할 때 사용
  부모요소 후손요소
  ex) div p
  div
    p
    section
      p

7. 동위(형제) 선택자
  선택자 ~ 형제요소  선택자와 동일한 위치 뒤에 있는 형제요소 모두선택
  선택자 + 형제요소  선택자와 동일한 위치 바로 뒤에 있는 형제요소 선택
  ex) p + section
    div
      p
      section
        p
      section
        p

8. 속성 선택자
  [속성]      속성을 포함
  [속성=값]   속성의 값과 완전히 일치하는 것
  [속성^=값]  속성의 값으로 시작하는 것
  [속성$=값]  속성의 값으로 끝나는 것
  [속성*=값]  속성의 값을 포함

  ex) [src*=png]
  1. <img src="test.jpg" alt="테스트이미지">
  2. <img src="test.png" alt="테스트">
  3. <img src="test.gif">

9. 반응, 상태, 링크 선택자
  :link     링크
  :hover    마우스를 올렸을 때
  :active   마우스로 클릭한 상태
  :visited  사이트를 방문한 상태
  :checked  체크가 된 상태
  :enabled  사용가능한 상태
  :disabled 사용불가능한 상태
  :focus    포커스가 적용된 상태
  :target   앵커의 타겟이 된 상태

10. 구조선택자
  :first-child    상위요소의 하위요소들 중 첫번째
  :last-child     상위요소의 하위요소들 중 마지막
  :nth-child(수열 또는 odd 또는 even) 상위요소의 하위요소들 앞에서()번째
  :nth-last-child(수열 또는 odd 또는 even) 상위요소의 하위요소들 뒤에서()번째
  :only-child    특정 하위요소 하나만을 포함(다른 요소 포함x)

  *(제이쿼리 선택자)
  :first         상위요소의 하위요소들 중 첫번째(동일한구조가 중첩되어있어도 하나만)
  :last          상위요소의 하위요소들 중 마지막(동일한구조가 중첩되어있어도 하나만)

  태그의 형태를 구분할 경우
  :first-of-type    상위요소의 하위요소들 중 첫번째
  :last-of-type     상위요소의 하위요소들 중 마지막
  :nth-of-type(수열 또는 odd 또는 even) 상위요소의 하위요소들 앞에서()번째
  :nth-last-of-type(수열 또는 odd 또는 even) 상위요소의 하위요소들 뒤에서()번째
  :only-of-type    특정 하위요소 하나만을 포함(다른 요소 포함가능)

11. 문자선택자
  ::first-letter    요소의 첫번째 문자
  ::first-line      요소의 첫번째 줄 문자
  ::before          요소의 내용 앞
  ::after           요소의 내용 뒤

12. 부정선택자
  :not(선택자)      선택자가 아닌 것을 선택
  :not(.item)

13. 다른 선택자를 여러개 선택
  중복되는 속성과 값을 가지고 있는 요소의 경우 ,(쉼표)를 사용하여 선택
  h1{color:red;}
  h2{color:red;}
  p{color:red;}
  h1,h2,p{color:red;}

14. 복합 선택자
  선택자의 종류를 복합적으로 조합하여 사용 하는 것

  연습문제)
  div.demo > h1 span + a{color:red;}
  말로 풀어서 나열해보자
  div중에 클래스가 demo인 것을 선택을 하고
  자식 중 h1을 선택
  h1의 후손 중에 span을 선택하고 그 바로 뒤에있는 a를 최종선택

  ...
  <div class="demo">
    <h1>
      <span>아이콘</span>
      <a href="#">바로가기1</a>
      <a href="#">바로가기2</a>
    </h1>
    <h2></h2>
    <div></div>
  </div>
  ...

  div .demo h1 span ~ a{}
